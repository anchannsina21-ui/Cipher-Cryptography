1.4 ‚Äî Transposition Cipher

Unlike substitution ciphers that replace letters, transposition ciphers rearrange the positions of characters according to a key. The message is written into a grid row by row, and read column by column in the order defined by the key.

1
Write the key above columns

The key defines the column order. Sort the key letters alphabetically to get the read order.

2
Fill the grid row-by-row

Write the plaintext into rows under the key columns (pad with X if needed).

3
Read columns in alphabetical key order

The ciphertext is the concatenation of columns read in key order.
